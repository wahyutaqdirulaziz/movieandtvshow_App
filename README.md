
# Movie App
A movie app made in Flutter with  BLoC, Hive and the movie db API.

## Features
- Search feature
- Watchlist feature
- Now playing movies
- Popular movies
- Top rated movies
- Movie details
- Popular tv shows
- Top rated tv shows

## Installation
### 1. Clone the repo
```bash
git clone https://github.com/wahyutaqdirulaziz/movieandtvshow_App.git
cd flutter-movie-app
```
### 2. Run pub get
```bash
flutter pub get
```
### 3. Add your API key into constant_helpers file
```bash
apiKey = 'YOUR_API_KEY';
```
### 4. Run the app
```bash 
flutter run
```
